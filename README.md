# Simple-patient-health-monitoring-system

## Overview
The Simple Patient Health Monitoring System is an IoT-based healthcare project designed to monitor essential health parameters of a patient in real time. The system uses sensors and a microcontroller to measure body temperature, heart rate, and other vital signs, helping in continuous patient observation and early detection of health issues.

## Features
- Real-time patient health monitoring
- Heart rate and temperature measurement
- Display of readings on LCD/Serial Monitor
- Alert generation for abnormal conditions
- Low-cost and easy-to-use system

## Components Used
- ESP32/Arduino
- Pulse Sensor
- Temperature Sensor (LM35/DHT11)
- LCD Display
- Buzzer
- Jumper Wires and Breadboard

## Applications
- Hospitals
- Home healthcare monitoring
- Elderly patient monitoring
- Rural healthcare systems

## Advantages
- Continuous health tracking
- Portable and compact
- Cost-effective
- Easy implementation using IoT

## Future Scope
The system can be enhanced by integrating cloud storage, mobile applications, and wireless communication for remote patient monitoring and emergency alerts.
